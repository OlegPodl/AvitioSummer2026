# Тест-кейсы для UI
Все тест-кейсы прописаны для следующих браузеров: Microsoft Edge, Google Chrome, Firefox

## Тест-кейсы для десктопной версии браузера
### 1. Проверка корректной работы фильтра "Диапазон цен":
    1. TEST_PRICE_FILTER_01 - Проверка валидного диапазона цен:
        * Шаги:
          1.  Ввести минимальный диапазон 1 000
          2.  Ввести максимальный диапазон 10 000
        * Ожидаемый результат:
            Показаны объявления только с ценой в диапазоне от 1000 до 10 000
    2. TEST_PRICE_FILTER_02 - Проверка отрицательного диапазона цен:
        * Шаги:
           1.  Ввести минимальный диапазон -10 000
           2.  Ввести максимальный диапазон -1 000
        * Ожидаемый результат:
           1. Сообщение об ошибке
           2. Пустой результат поиска
    3. TEST_PRICE_FILTER_03 - Проверка буквенных значений в диапазоне цен:
        * Шаги:
            1.  Попытка ввести минимальный диапазон abc
            2.  Попытка ввести максимальный диапазон def
        * Ожидаемый результат:
            Значения диапазонов остаются пустыми
    4. TEST_PRICE_FILTER_04 - Проверка символьных значений в диапазоне цен:
        * Шаги:
            1.  Попытка ввести минимальный диапазон @@@
            2.  Попытка ввести максимальный диапазон %%%
        * Ожидаемый результат:
            Значения диапазонов остаются пустыми
    5. TEST_PRICE_FILTER_05 - Проверка: минимальный диапазон превышает максимальный:
        * Шаги:
            1.  Ввести минимальный диапазон 10 000
            2.  Ввести максимальный диапазон 1 000
        * Ожидаемый результат:
            1. Сообщение об ошибке
            2. Пустой результат поиска
    6. TEST_PRICE_FILTER_06 - Проверка пустого диапазона цен:
        * Шаги:
            1.  Ввести минимальное значение ""
            2.  Ввести максимальное значение ""
        * Ожидаемый результат:
            Показаны все объявления

### 5. Проверка страницы статистики:
    1. TEST_TIMER_CONTROL_CONTAINER_01 - Проверка обновления статистики по истечению таймера
        * Шаги:
            1. Считать значение таймера
            2. Подождать его окончания
        * Ожидаемый результат:
            Статистика обновлена
    2. TEST_TIMER_CONTROL_CONTAINER_02 - Проверка ручного обновления статистики
        * Шаги:
            1. Нажать на кнопку ручного обновления статистики
        * Ожидаемый результат:
            Статистика обновлена
    3. TEST_TIMER_CONTROL_CONTAINER_03 - Проверка остановки таймера
        * Шаги:
            1. Нажать на кнопку остановки таймера
        * Ожидаемый результат:
            Таймер прекращает отсчет
    4. TEST_TIMER_CONTROL_CONTAINER_04 - Проверка запуска таймера после остановки
        * Шаги:
            1. Нажать на кнопку остановки таймера
            2. Убедится что таймер прекратил отсчет
            3. Нажать на кнопку продолжения отсчета таймера
        * Ожидаемый результат:
            Таймер продолжил отсчет, по окончании таймера статистика обновлена
### 6. Проверка мобильной версии сайта:
